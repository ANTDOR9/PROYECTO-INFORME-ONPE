# PROYECTO-INFORME-ONPE · SED-2026

Simulación del Sistema de Escrutinio Digital de la ONPE para las Elecciones Generales 2026 — Segunda Vuelta del 7 de junio.

Desarrollado como tarea integradora final del módulo **HT-02 · Metodologías Ágiles** — ISMA SENATI.

---

## ¿Qué es?

Dos proyectos en uno:

- **Sistema ONPE** — réplica del portal electoral con registro de actas, resultados en tiempo real y monitor de mesas
- **Informe UML** — análisis arquitectónico con 5 diagramas UML del sistema modelado

## Tecnologías

HTML · CSS · JavaScript vanilla · Chart.js · GSAP · localStorage · Sin backend
